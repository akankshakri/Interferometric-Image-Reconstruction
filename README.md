# VAE-Based Interferometric Image Reconstruction

## Overview
This project uses a Variational Autoencoder (VAE) to reconstruct astronomical images from sparse interferometric visibility measurements.

## Features
- Visibility data preprocessing
- VAE encoder-decoder architecture
- Image reconstruction
- Evaluation using PSNR, SSIM, and Separation Error

## Dataset
- Supergiant B[e] Star dataset
- Synthetic M87 dataset

## Requirements
Python 3.10+
TensorFlow/PyTorch
NumPy
Matplotlib

## Results
The model reconstructs astronomical source images from visibility measurements and compares them with ground-truth images.
